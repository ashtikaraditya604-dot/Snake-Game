# 🐍 Classic Snake Game

A browser-based Snake Game built using **HTML, CSS Grid, and JavaScript**, featuring smooth gameplay, real-time rendering, and increasing difficulty levels.

---

## 🚀 Features

* 🎮 Interactive snake movement using keyboard controls
* ⚡ Real-time rendering using DOM manipulation
* 💥 Collision detection (walls & self)
* 📊 Score tracking system
* 📈 Progressive difficulty for engaging gameplay
* 📱 Responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS Grid
* JavaScript (DOM Manipulation)

---

Snake-Game/

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start playing 🎉
